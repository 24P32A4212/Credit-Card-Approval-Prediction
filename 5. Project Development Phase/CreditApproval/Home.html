Project Architecture & Scope

Financial systems require scalable alternatives to manual checks. This platform converts granular applicant demographics and historical credit balances into high-fidelity evaluations using a serialized predictive engine. The back-end handles data parsing, cleaning feature offsets, and addresses class imbalances dynamically before computing eligibility indices.

Implemented Engine Algorithms:
Logistic Regression Random Forest Classifier Gradient Boosting (XGBoost) Decision Tree Classifier

Model Engine Status

  • Preprocessing: Categorical Label Encoding mapped cleanly to Python integer vectors.
  • Feature Adjustments: Native negative-offset handling applied across DAYS_BIRTH & DAYS_EMPLOYED.
  • Platform Integration: Serialization via Joblib/Pickle connected to real-time Flask triggers.
