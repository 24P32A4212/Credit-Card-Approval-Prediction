
💳 Credit Card Approval Prediction Using Machine Learning

📌 Project Overview

The Credit Card Approval Prediction project is a Machine Learning-based web application that predicts whether a customer's credit card application is likely to be Approved or Rejected based on the applicant's financial and personal details.

The main objective of this project is to help banks and financial institutions automate the credit approval process, reducing manual effort and improving decision-making accuracy.

---

🚀 Features

- User-friendly web interface
- Predicts credit card approval instantly
- Data preprocessing and feature engineering
- Multiple Machine Learning algorithms tested
- Random Forest selected as the best-performing model
- Fast, accurate, and easy to use

📊 Dataset

The dataset contains applicant information such as:

- Gender
- Age
- Annual Income
- Employment Status
- Education
- Marital Status
- Property Ownership
- Car Ownership
- Family Members
- Housing Type
- Occupation
- Credit History

These attributes are used to determine whether a customer is eligible for a credit card.

▶️ Installation

Clone the repository:

git clone https://github.com/24P32A4212/Credit-Card-Approval-Prediction.git

Go to the project directory:

cd Credit-Card-Approval-Prediction

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open your browser and visit:

http://127.0.0.1:5000/
